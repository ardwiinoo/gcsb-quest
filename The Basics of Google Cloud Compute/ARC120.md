# The Basics of Google Cloud Compute: Challenge Lab (ARC120)

## Task 1. Create a Cloud Storage bucket

Create bucket
```bash
gsutil mb -l US gs://[PROJECT_ID]-bucket
```

Verify Bucket

```bash
gsutil ls
```

## Task 2. Create and attach a persistent disk to a Compute Engine instance

Create compute engine instance named **my-instance**

```bash
gcloud compute instances create my-instance \
  --zone=[PROJECT_ZONE] \
  --machine-type=e2-medium \
  --image-family=debian-12 \
  --image-project=debian-cloud \
  --boot-disk-size=10GB \
  --boot-disk-type=pd-balanced \
  --tags=http-server
```

Create a new persistent disk named **mydisk** with a size of 200GB

```bash
gcloud compute disks create mydisk \
  --zone=[PROJECT_ZONE] \
  --size=200GB \
  --type=pd-standard
```

Attach the persistent disk to the instance

```bash
gcloud compute instances attach-disk my-instance \
  --disk=mydisk \
  --zone=[PROJECT_ZONE]
```

Verify instance and disks

```bash
gcloud compute instances describe my-instance \
  --zone=[PROJECT_ZONE]
```

## Task 3. Install a NGINX web server

SSH into the instance

```bash
gcloud compute ssh my-instance --zone=[PROJECT_ZONE]
```

Update the OS

```bash
sudo apt update
sudo apt upgrade -y
```

Install NGINX

```bash
sudo apt install nginx -y
```

Start and Enable NGINX

```bash
sudo systemctl start nginx
sudo systemctl enable nginx
```

Verify NGINX

```bash
sudo systemctl status nginx
```
